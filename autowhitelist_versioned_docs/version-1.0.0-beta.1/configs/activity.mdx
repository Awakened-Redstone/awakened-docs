#### Possible activity values are:

- **NONE** (Doesn't change the bot status, for when another application is setting it)
- **RESET** (Hides the bot status)
- <span className={"text-green-500"}>**PLAYING** (Sets the bot status to "Playing ...")</span>
- **STREAMING** (Sets the bot status to "Streaming ..."
- **LISTENING** (Sets the bot status to "Listening to ...")
- **WATCHING** (Sets the bot status to "Watching ...")
